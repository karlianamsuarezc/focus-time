# FocusFlow ⏳

Un temporizador de productividad minimalista diseñado bajo principios de ingeniería escalable y una estética **Soft & Chic**.

## 🚀 Sobre el Proyecto
Este proyecto es una aplicación móvil desarrollada con **React Native** y **Expo**. 
El objetivo es crear una herramienta de gestión de tiempo (Pomodoro/Timer) que combine funcionalidad técnica avanzada con una interfaz limpia, neutra y profesional.

### 🛠️ Tech Stack & Metodología
* **Framework:** React Native (Expo SDK)
* **Lenguaje:** JavaScript / TypeScript
* **Arquitectura:** Estructura modular inspirada en *Antigravity* (Separación de lógica, core y componentes).
* **Estilo:** Minimalismo Soft & Chic (Paleta de neutros, tipografía limpia).

## 📁 Estructura de Carpetas (Antigravity Lite)
```text
/src
 ├── /core         # Temas, constantes y configuración global.
 ├── /modules      # Lógica de negocio (Timer, Estadísticas, etc).
 ├── /shared       # Componentes de UI reutilizables.
 └── /services     # Llamadas a API o persistencia local.
